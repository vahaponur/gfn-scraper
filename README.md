# GeForce NOW Scraper

Gets all games currently avaliable in GeForce NOW. With Platforms and Publishers

## Installation

Github CLI

```bash
gh repo clone vahaponur/gfn-scraper
```

npm
```bash
npm i gfn_scraper
```

## Usage

```
import getGfnGames from "./gfn_scraper.js";

let games = await getGfnGames()
```
